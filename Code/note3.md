## append vari in print statment

1. use '+' for string
- Eg: var name = "Ashish"
	fmt.Println("Your name: " + name)

2. use ',' for string, int 
- It's automatically not append one space in prefix of variable
- Eg: var ageTeen = 18
	fmt.Println("Teen age:", ageTeen)
